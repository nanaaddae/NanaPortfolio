Omega Bookstore App

Manage your book collection effortlessly with the Omega Bookstore App! This Python application, equipped with a sleek Tkinter-based GUI, offers a seamless experience for adding, viewing, editing, and deleting book entries in a SQLite database.

Key Features:

  • View All Books: Instantly access a detailed list of all your books.
  • Add to Database: Effortlessly add new books with a few clicks.
  • Delete Book: Streamline book removal using ISBN for accuracy.
  • Edit Book: Easily modify book details to keep information up-to-date.
  • Search and Filter: Quickly find specific books using a search bar and filter options.
  • Sorting Options: Enhance flexibility by sorting books based on various criteria.

Technology Stack:

  • Python 3.x
  • Tkinter Library
  • SQLite3 Database

GitHub Pages Integration:

Share your book collection with the world! The Omega Bookstore App seamlessly integrates with GitHub Pages, providing a user-friendly web interface for easy access.

How to Use:

  1. Clone the repository.
  2. Run the application using Python.
  3. Enjoy an intuitive GUI for efficient book management.

Overview

This real-time chat application is a modern, scalable messaging platform that enables instant communication between users in private or password-protected group rooms. Built with Spring Boot on the backend and React on the frontend, it leverages WebSocket with STOMP protocol for bidirectional, low-latency messaging while maintaining strong security through JWT-based authentication.

Key Impact: Supports 100+ concurrent users with features like live typing indicators, online presence tracking, and seamless message delivery—ideal for team collaboration, community chats, or customer support tools.

Core Features & Implementation

1. Real-Time Messaging Engine with WebSocket/STOMP

Engineered the core real-time communication layer using Spring Boot's WebSocket support and STOMP (Simple Text Oriented Messaging Protocol) over SockJS fallback for broad browser compatibility. This enables instant message broadcasting, typing indicators (via periodic "user is typing" events), presence tracking (online/offline status), and live updates without page refreshes.

Messages are routed efficiently through destinations like /topic/public (broadcast) and /user/queue/private (targeted delivery), ensuring reliable delivery even under network interruptions.

2. Secure Authentication & Protected Rooms

Implemented JWT (JSON Web Tokens) authentication for user sessions, integrated with Spring Security to protect both REST endpoints and WebSocket connections. Users register/login via secure APIs, receive tokens, and must provide valid tokens to connect to WebSocket or join rooms.

Added password-protected chat rooms: Room creators set optional passwords, and joining requires validation—preventing unauthorized access while supporting public/open rooms for casual use.

3. RESTful API & Responsive React Frontend

Built a comprehensive REST API with 15+ endpoints for user management, room creation/joining, message history retrieval, and profile updates using Spring Boot and Spring Data JPA.

On the frontend, developed a responsive Single Page Application (SPA) in React featuring:

  • Typing indicators and user presence (online/offline badges)
  • Room list sidebar with search and creation
  • State management via React Context API + Axios for API calls + SockJS/STOMP client for WebSocket

Technical Challenges & Solutions

  • Securing WebSocket connections → Extended Spring Security to intercept and validate JWT in WebSocket handshake headers.
  • Message persistence → Stored chat history in database for reload/resume functionality while keeping real-time delivery lightweight.

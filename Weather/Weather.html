Weather Application

This project is a simple weather application that allows users to check the current weather conditions based on a provided zip code. It consists of a frontend built with React and a backend built with Spring Boot.

Features:

  • Accepts user input for zip code
  • Fetches weather data from a Spring Boot backend
  • Displays current weather information, including city name, temperature, and conditions

Technologies Used:

  • React for the frontend
  • Spring Boot for the backend
  • Bootstrap for styling

How to Use:

Simply enter a valid zip code, click the "Get The Weather!" button, and the application will fetch and display the current weather information.
